The Mandelbrot Set

The Mandelbrot set is a set of points in the complex plane whose boundary forms a fractal. It is defined by the iterative formula:

zn+1 = zn2 + c

For each point c, we start with z0 = 0. If the sequence remains bounded (does not escape to infinity), the point belongs to the set and is colored black. If it escapes, it is colored based on how quickly it diverges.

This implementation uses WebGL for GPU acceleration, allowing for real-time fluid interaction even at deep zooms.
